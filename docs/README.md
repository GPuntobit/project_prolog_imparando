# Cartella Materiale Didattico

Questa cartella contiene i file PDF con materiale didattico, esempi, teorie e riferimenti su Prolog.

## Come funziona

I file PDF presenti in questa cartella vengono automaticamente consultati dal sistema AI prima di rispondere a qualsiasi domanda. Questo garantisce che le risposte siano allineate con il materiale didattico fornito.

## Come aggiungere PDF

1. Copia i tuoi file PDF in questa cartella (`docs/`)
2. I file verranno automaticamente letti e considerati nelle risposte
3. Non è necessario configurare nulla: il sistema li rileva automaticamente

## Note

- I PDF vengono letti automaticamente ad ogni interazione
- Il contenuto dei PDF viene integrato nelle spiegazioni e negli esempi
- Assicurati che i PDF siano in formato leggibile e non protetti da password
