# Cartella Materiale Didattico

Questa cartella contiene i file PDF con materiale didattico, esempi, teorie e riferimenti su Prolog.

## Come funziona

I file PDF presenti in questa cartella vengono automaticamente consultati dal sistema AI prima di rispondere a qualsiasi domanda. Questo garantisce che le risposte siano allineate con il materiale didattico fornito.
